# GitHub Activity Tracker 🚀

A modern, full-stack web application to track and visualize GitHub user activity in real-time.

## Features

- 🔍 Search any GitHub user by username
- 📊 Display recent activity (commits, issues, stars, forks, PRs, etc.)
- 🎨 Modern, animated UI with dark theme
- ⚡ Real-time data fetching from GitHub API
- 📱 Fully responsive design
- 🎭 Smooth animations with Framer Motion

## Tech Stack

**Frontend:**
- React 18
- Framer Motion
- Axios
- Custom CSS

**Backend:**
- Node.js
- Express
- Axios
- CORS

## Local Development

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository
```bash
git clone https://github.com/a1zen77/github-activity
cd github-activity-app
```

2. Install backend dependencies
```bash
cd backend
npm install
```

3. Install frontend dependencies
```bash
cd ../frontend
npm install
```

### Running the Application

1. Start the backend (in one terminal)
```bash
cd backend
npm run dev
```

2. Start the frontend (in another terminal)
```bash
cd frontend
npm start
```

3. Open http://localhost:3000 in your browser
